开源 Agent 运行时

Eva-CLI

面向多 Agent 自动化的受控运行时

Eva-CLI 用 Rust 承载可靠运行时边界，用 Lua 承载可热更新 Agent 行为，并通过统一 Topic EventBus 连接外部 Agent、MCP、Skill、记忆、知识库与硬件能力。

Rust 运行时边界 Lua 热更新 Topic EventBus

执行模型 受控自动化链路
  1. Rust Core 托管 policy、记忆、恢复和进程边界。
  2. Lua Agents 热更新 Agent 行为，同时保持能力访问显式可控。
  3. Adapters 通过权限边界接入 CLI、MCP、HTTP、Skill 与硬件。

V1.17.6 仍保持 V1.11.5-alpha runtime 版本，同时把 V1.x closure release gate 覆盖到更宽的证据线：daemon control、provider supervision gate、restore mutation/rollback evidence、service-manager abstraction、hardware safety、memory 与 observability policy、operator execution fields、高风险 text summary、public JSON contract validation 和 external blocker 记录。
