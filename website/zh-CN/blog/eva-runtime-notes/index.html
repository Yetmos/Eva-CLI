Eva 运行时笔记

关于 Eva-CLI 运行时边界的设计笔记。

发布： 2026-06-17 分类： 运行时

Eva-CLI 用 Rust 承载运行时所有权，用 Lua 承载可热更新行为。这个边界让恢复、策略、记忆归属和能力控制都有明确的负责层。

运行时边界

核心设计点是不要把策略和恢复逻辑放进可热更新脚本里。Lua 可以描述 Agent 行为，但 Rust 运行时仍然负责校验能力、管理进程状态，并决定事件如何在系统中流动。

为什么重要

受控运行时边界让系统更容易检查，也更容易安全演进。Adapter、MCP server、Tool、Skill、记忆和硬件能力都可以通过显式权限接入，而不是依赖脚本的隐式访问。
