Eva-CLI

为多 Agent 协作而设计的可扩展运行时：用 Rust 承载稳定核心，用 Lua 交付热更新能力，并通过统一事件总线连接外部 Agent、MCP、Skill 与硬件设备。

Eva 开源核心项目，为 EvaLauncher 等产品提供可控的 Agent Runtime 基础。
