# Eva-CLI V1.5.0 发布说明

日期：2026-07-04

Eva-CLI V1.5.0 是已实现 1.x 源码发布面的发布加固 checkpoint。它新增可执行的发布就绪检查，同时保持 V1.0 到 V1.4 的命令契约。

## 新增

- 新增 `eva-release` crate，提供 release readiness、security review、performance baseline、migration guide 和 compatibility policy 契约。
- 新增 CLI 命令：
  - `eva release check`
  - `eva release security`
  - `eva release perf`
  - `eva release migration`
- CI 在 Windows、Linux、macOS smoke 中覆盖 release 命令组。
- 新增 V1.5 发布加固文档、迁移指南、兼容策略和发布说明。

## 稳定

- V1.0-V1.4 命令继续可用，JSON envelope 保持稳定。
- `restore plan`、`upgrade check` 和 `hardware bind` 继续保持 plan-first、非破坏性。
- 安全 finding 显式记录，包括真实 restore 与进程 handoff 的后续风险。

## 明确非目标

- 不提供安装包或签名发布 artifact。
- 不启动真实 Supervisor 进程。
- 不提供 destructive restore apply path。
- 不启动真实 MCP server，不打开硬件 raw I/O。
- 不提供生产级 benchmark harness。
