## 此 tag 包含

- `LuaShadowLoader` 使用 resource limit 和 no-op capability host 执行 dry-run
  health check。
- `GenerationRouteGate` 在 candidate health 通过前让新 work 留在 active generation。
- `GenerationDrainEvidence` 记录 route/drain planning 和 in-flight count。
- `plan_generation_lifecycle_rollback()` 保留上一 healthy route，并输出 rollback
  audit evidence。
- 编译进代码的 `REL-LUA-HOT-RELOAD-001` 和 marker
  `lua_hot_reload_lifecycle_ready`。

## 未包含

此 tag 不会启动、drain 或替换生产 daemon。它提供 shadow、route、drain 和
rollback planning/evidence 边界，不是完整 daemon 驱动 hot reload、service-manager
集成、provider recovery、破坏性 apply 或签名分发。

## 复现发布

```powershell
git switch --detach v1.7.4-alpha

cargo clippy --workspace --all-targets -- -D warnings

cargo test -p eva-lua-host shadow_load
cargo test -p eva-scheduler generation
cargo test -p eva-lifecycle drain
cargo test -p eva-lifecycle rollback
./scripts/validate-version-management.ps1 -Tag v1.7.4-alpha
cargo run -- release check --output json
```
