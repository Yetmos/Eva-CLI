## 此 tag 包含

- EventLog、dead-letter 和 EventBus 的只读打开路径。
- `eva_runtime::inspect_durable_backend()` 和
  `DurableDiagnosticsReport`，报告 schema、layout、migration、event、
  dead-letter 和 pending-redrive 计数。
- `eva inspect durable --durable-backend <path>` 的 text 和
  `inspect.durable` JSON 输出。
- 编译进代码的 `REL-DURABLE-DIAGNOSTICS-001` 和
  `inspect-durable.json` workflow evidence。

## 未包含

此 tag 不包含完整 runtime audit wiring、scheduler backoff dispatcher、provider
process recovery、破坏性 apply、生产 database 或签名分发。诊断命令只读。

## 复现发布

```powershell
git switch --detach v1.6.5-alpha
cargo fmt --check
cargo clippy --workspace --all-targets -- -D warnings
cargo test --workspace
./scripts/validate-version-management.ps1 -Tag v1.6.5-alpha
$durable = Join-Path $env:TEMP "eva-v1.6.5-smoke"
cargo run -- run --example basic --task-id req-v165 --durable-backend $durable --output json
cargo run -- inspect durable --durable-backend $durable --output json
cargo run -- release check --output json
```
