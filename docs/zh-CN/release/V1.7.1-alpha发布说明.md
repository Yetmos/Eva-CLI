## 此 tag 包含

- `MluaVmAdapter` 使用 vendored Lua 5.4，只加载 table、string、utf8 和 math 库。
- 支持 return/global/table 脚本形态的真实 `on_event` 执行。
- 受控 event/context 数据，不暴露 file、socket、process、memory service 或
  provider handle。
- 稳定的 `lua_syntax_error` 和 `lua_runtime_error` 映射，不泄露 host path。
- legacy 受控脚本 compatibility fallback 和编译进代码的
  `REL-LUA-VM-EXECUTION-001`。

## 未包含

此 tag 不包含 `ctx.tools`、host log/audit binding、执行预算、shadow load、
generation routing 或 live provider execution。后续 V1.7 tag 补齐了除 live
provider execution 外的这些受控 runtime 边界。

## 复现发布

```powershell
git switch --detach v1.7.1-alpha

cargo run -- release check --output json
```
