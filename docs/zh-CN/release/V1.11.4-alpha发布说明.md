# Eva-CLI V1.11.4 Alpha 发布说明

状态：alpha 预发布
Tag：`v1.11.4-alpha`

Eva-CLI V1.11.4-alpha 完成 CLI command module split 切片。它在保持公开
命令契约稳定的前提下，把 release、version、config、adapter、MCP、discovery、
memory、observability、task、skill、hardware、backup、snapshot、restore、upgrade、
inspect 和 doctor 命令处理拆入更聚焦的模块。

## 重点

- 将主要命令 parser、executor 和 formatter 从 `run.rs` 拆入
  `crates/eva-cli/src/run/*.rs` 命令模块。
- 保持共享 JSON envelope、trace 字段、exit code 映射和命令输出形状不变。
- 继续保留 artifact provenance、distribution smoke、external scanner、benchmark
  evidence、restore apply gate 和 supervisor handoff gate 等 release evidence 门禁。
- 将当前发布元数据提升到 `1.11.4-alpha`，在补齐新的 `emit`、`agent` 和
  `capability` 命令前固定 CLI 模块拆分检查点。

## 兼容性

V1.11.4-alpha 预期不改变已有公开 CLI 命令行为。它不新增 destructive restore
mutation、生产 service manager handoff、真实硬件 I/O 或签名安装器凭据。

## 验证

- `cargo fmt --check`
- `cargo test -p eva-cli`
- `cargo run -- --version`
- `cargo run -- release check --output json`
- `scripts/validate-version-management.ps1 -Tag v1.11.4-alpha`

## 制品

- GitHub Release 源码压缩包。
- release workflow 完成后生成 `release-evidence-v1.11.4-alpha` artifact。
- package 发布成功时，GHCR 容器 package 为 `ghcr.io/yetmos/eva-cli:1.11.4-alpha`。

`emit`、`agent` 和 `capability` 命令补齐保留为下一段 V1.11.5 切片。
