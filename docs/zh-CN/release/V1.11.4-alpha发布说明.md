## 此 tag 包含

- 将 release、version、config、adapter、MCP、discovery、memory、
  observability、task、skill、hardware、backup、snapshot、restore、upgrade、
  inspect 和 doctor 命令处理拆入聚焦的 `run/*.rs` 模块。
- 保持共享 JSON envelope、trace 字段、退出码映射和输出格式集中管理。
- 保留既有编译期 release evidence 声明，并新增 marker
  `release_distribution_cli_split_v1.11.4`。

## 未包含

该拆分以保持行为为目标，不新增 `emit`、`agent` 或 `capability` 命令；这些命令
由下一个 tag 提供。它也不包含破坏性 restore mutation、生产 service-manager
handoff、真实硬件 I/O、provider supervision 或签名凭据。

## 复现发布

```powershell
git switch --detach v1.11.4-alpha
cargo fmt --check
cargo clippy --workspace --all-targets -- -D warnings
cargo test --workspace
cargo test -p eva-cli
./scripts/validate-version-management.ps1 -Tag v1.11.4-alpha
cargo run -- --version
cargo run -- release check --output json
```
