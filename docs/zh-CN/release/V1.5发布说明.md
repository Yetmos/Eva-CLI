## 此 tag 包含

- 新增 `eva-release` crate，以及 `eva release check`、`security`、`perf`
  和 `migration` 命令。
- 保持 V1.0-V1.4 命令名、JSON envelope 和退出码契约。
- restore、upgrade 和 hardware 操作保持 plan-first、非破坏性。
- release 命令组增加 Windows、Linux 和 macOS smoke 覆盖。

`release check` 评估该 tag 编译进代码的声明和内置证据。`ready` 不表示它执行了
CI、外部扫描或生产发布。

## 未包含

此 tag 不包含打包安装器、签名制品、生产 benchmark、真实 Supervisor 进程管理、
破坏性 restore apply、真实 MCP server transport 或硬件 raw I/O。

## 复现发布

```powershell
git switch --detach v1.5.0
cargo fmt --check
cargo clippy --workspace --all-targets -- -D warnings
cargo test --workspace
cargo run -- --version
cargo run -- release check --output json
cargo run -- release security --output json
cargo run -- release perf --output json
cargo run -- release migration --output json
./scripts/validate-i18n.ps1
```
