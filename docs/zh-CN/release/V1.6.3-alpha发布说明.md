## 此 tag 包含

- `FileSystemTaskStateStore::from_durable_layout`，以及 basic run 和 task
  diagnostics 使用的增量 `--durable-backend` CLI 路径。
- `FileSystemAuditSink`，支持按 span、request、event、correlation 和
  causation identifier 查询。
- artifact metadata v2，包含 digest、size、content type 和 retention 字段；
  legacy metadata 仍可读取，损坏 metadata 返回 `ErrorKind::Conflict`。
- 编译进代码的 `REL-DURABLE-STORES-001` gate。

## 未包含

此 tag 不包含 crash recovery、scheduler dispatch、runtime audit wiring 或
durable diagnostics。该 gate 也不是外部 durability 或生产发布测试。
