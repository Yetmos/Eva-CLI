# Eva-CLI V1.6.3 Alpha 发布说明

状态：alpha 预发布
Tag：`v1.6.3-alpha`

Eva-CLI V1.6.3-alpha 完成 V1.6.3 durable task/audit/artifact store 切片。这个
版本在 V1.6.1 durable backend 布局和 V1.6.2 durable EventBus redrive 基线上，
把 task snapshot 接入 durable backend，新增 durable audit record，并补齐
filesystem artifact metadata hardening。

## 重点

- 新增 `FileSystemTaskStateStore::from_durable_layout`，`run --example basic`
  和 `task status/logs/cancel` 可通过 `--durable-backend <path>` 使用 durable
  backend 的 `tasks/` 目录。
- 新增 `FileSystemAuditSink` 和 `AuditRecord`，把 durable audit record 保存到
  `audit/` 目录，记录 action、outcome、trace entries、message、fields 和毫秒时间戳。
- durable audit record 支持按 span、request、event、correlation、causation id 查询。
- filesystem artifact metadata 升级为 v2，记录 key、digest、size、content type、
  retention policy 和可选 retain-until timestamp。
- 继续兼容 legacy artifact metadata；content type 或 retention metadata 损坏时返回稳定
  `Conflict`。
- `release check` 新增 `REL-DURABLE-STORES-001` 门禁。

## 兼容性

V1.6.3-alpha 不改变公开 CLI 命令名、JSON envelope 形状、exit code 或既有
V1.5/V1.6 release 命令。新的 durable backend 参数是增量能力，旧 artifact metadata
仍可读取。

它仍是 alpha 检查点，因为 runtime crash recovery、scheduler backoff dispatch、
runtime audit wiring、durable smoke 诊断、签名安装器和 destructive apply gate 仍在后续节点。
