# Eva-CLI V1.6.1 Alpha 发布说明

状态：alpha 预发布
Tag：`v1.6.1-alpha`

Eva-CLI V1.6.1-alpha 启动 V1.6 durable runtime/storage 线。这个版本先落地
durable backend schema 与 migration lock 基线，后续 EventBus、task、audit 和
artifact store 会复用同一持久化布局边界。

## 重点

- 新增 `FileSystemDurableBackend`，通过 `backend.manifest` 固定 schema version
  `1`、layout version `eva.durable.v1`，并创建稳定的 `events/`、`state/`、
  `tasks/`、`audit/`、`artifacts/` 目录。
- 新增读写打开时的 `migration.lock` 迁移锁；如果读写打开在校验阶段失败，也会释放锁文件。
- 新增只读打开校验路径：不会创建文件，也不会获取迁移锁。
- 保留 `InMemoryDurableBackend`，供测试和后续兼容路径继续使用。
- 在 V1.x 真实运行时能力补齐实施计划中把 V1.6.1 标记为完成。

## 兼容性

V1.6.1-alpha 不改变公开 CLI 命令名、JSON envelope 形状、exit code 或既有
V1.5 发布加固命令。它仍是预发布版本，因为完整 V1.6 durable EventBus、
task/audit store、recovery coordinator 和 durable smoke gate 仍在后续节点中。
