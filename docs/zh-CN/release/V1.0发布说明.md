## 此提交包含

- `eva --version` 和 `eva version --output json` 提供 V1.0 版本输出。
- `RuntimeBuilder::in_memory_v10()` 将内存运行时标识为
  `in_memory_v1.0`，generation 为 `basic-v1.0`。
- `doctor`、`config validate`、`inspect`、`run --example basic` 和 task
  status/log/cancel 命令构成受支持的 CLI 边界。
- basic 示例与 CI 覆盖 JSON 成功/错误 envelope、trace 字段、结构化错误和退出码。

## 未包含

此检查点不提供外部 Adapter/MCP/Skill/Hardware 执行、真实 Lua VM、durable task
存储、daemon、supervisor 进程管理、backup/restore 执行、签名制品、安装器或
GitHub Release。
