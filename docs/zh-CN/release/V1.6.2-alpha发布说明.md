## 此 tag 包含

- `events/log` 下的 `FileSystemEventLog`，event 和 delivery metadata 可在重启后读取。
- `DurableEventBus` 和 `events/dead_letters` 下的
  `FileSystemDeadLetterStore`。
- 使用 `:replay-N` 的 redrive 子事件 ID，以及持久化的 retry timing 字段。

## 未包含

`retry_delay_ms` 和 `next_attempt_after_ms` 是兼容 metadata，不是延迟调度器。
此 tag 也不包含 durable task/audit store、runtime recovery 或 durable diagnostics；
这些切片由后续 V1.6 tag 提供。

## 复现发布

```powershell
git switch --detach v1.6.2-alpha
cargo fmt --check
cargo clippy --workspace --all-targets -- -D warnings
cargo test --workspace
cargo test -p eva-eventbus -p eva-storage
./scripts/validate-version-management.ps1 -Tag v1.6.2-alpha
cargo run -- --version
cargo run -- release check --output json
```
