# Eva-CLI V1.5.1 发布说明

状态：正式发布
Tag：`v1.5.1`

Eva-CLI V1.5.1 是 V1.5 发布加固线的 patch release。它保持 V1.5 CLI 和 JSON
契约稳定，同时通过 GHCR 启用 GitHub Packages 交付通道。

## 重点

- GitHub Release workflow 会把 release 容器镜像发布到 `ghcr.io/yetmos/eva-cli`。
- `release-evidence/package-ghcr.json` 会记录 GHCR package digest、tags、source tag、source SHA 和 package URL。
- stable release 可以更新 `latest` 容器 tag；prerelease tag 不会更新 `latest`。
- 保持现有 `v1.5.0` 源码发布为不可变历史，不为了回填 package metadata 重新发布它。
- 加固发布文档使用的 GitHub Pages artifact 路径。

## 兼容性

V1.5.1 不引入破坏性的 CLI、配置、JSON 或文档路径变更。

## 制品

- GitHub Release 源码压缩包。
- Workflow `release-evidence-v1.5.1` artifact。
- GHCR 容器 package：`ghcr.io/yetmos/eva-cli:1.5.1`。

签名安装器、系统包管理器 package 和 provenance bundle 仍属于后续 release 范围。
