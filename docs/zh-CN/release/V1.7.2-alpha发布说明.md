## 此 tag 包含

- 只读 `ctx.request`、`ctx.trace` 和 `ctx.memory` table；移除全局
  `rawset` 以保护 snapshot 边界。
- `ctx.host.log` 和 `ctx.host.audit` 生成 `LuaHostObservation` record。
- `ctx.tools.call` 通过 `CapabilityHostApi` 调用，不暴露 raw provider、file、
  socket、process、memory、knowledge 或 audit handle。
- 受限 JSON-compatible 请求/响应转换，并拒绝 unknown 或 disabled capability。
- 编译进代码的 `REL-LUA-HOST-BINDINGS-001`。

## 未包含

capability host 边界不是生产 provider supervision。此 tag 也不包含 timeout、
instruction/memory budget、shadow load、generation swap、rollback 执行或破坏性 apply。
