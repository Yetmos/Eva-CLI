# Eva-CLI V1.5 Release Acceptance

Date: 2026-07-06
Release tag: `v1.5.0`
Release URL: <https://github.com/Yetmos/Eva-CLI/releases/tag/v1.5.0>

## Decision

Eva-CLI V1.5.0 is already published as a GitHub-managed source release.
The public annotated tag `v1.5.0` exists on GitHub and resolves to commit
`74d85e7da58ac40ef5d30b38e2844dee503a44c0`.

The tag was not recreated during this acceptance pass. The project release plan
requires public tags to remain immutable after a GitHub Release is visible.
Documentation and Pages fixes that landed after the tag are tracked on `main`
and do not change the immutable V1.5.0 source-release record.
