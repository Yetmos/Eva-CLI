# V0.1 Baseline Acceptance

This page is the English entry for the Eva-CLI V0.1 baseline acceptance record.
The full-detail source is:
[V0.1 项目基线验收记录](../../zh-CN/planning/V0.1项目基线验收记录.md).

V0.1 is a project baseline release. It does not claim a working runtime. It
confirms that the repository has the documentation entry points, Rust workspace,
crate boundaries, root binary shim, static configuration samples, JSON schemas,
and testable baseline needed for V0.2 and later implementation work.

Accepted V0.1 evidence:

- `docs/`, `wiki/`, and `website/` contain the documentation and publishing
  structure.
- The root `Cargo.toml` defines version `0.1.0` and a workspace with 19 crates.
- `src/main.rs` delegates to `eva-cli`.
- `config/` contains sample Eva config, Agents, Adapters, Capabilities, Routes,
  Policies, and JSON Schemas.
- `docs/zh-CN/planning/当前项目从零到完整实现实施计划.md` defines the versioned plan
  from V0.1 to V1.5.
- `cargo test --workspace` passes.

V0.2 should start with policy types, route and policy schema alignment,
observability field contracts, and continued workspace test coverage.
