# Hardware Adapters / 硬件适配器

## 中文

硬件 Adapter 使用 `transport: hardware`，并声明设备总线、匹配规则、身份、协议、热插拔和设备级 limits。全局硬件权限边界位于 `config/policies/hardware.yaml`。

## English

Hardware Adapters use `transport: hardware` and declare device bus, match rules, identity, protocol, hotplug behavior, and device-level limits. Global hardware permission boundaries live in `config/policies/hardware.yaml`.
