# Capabilities / 能力定义

## 中文

能力定义描述 Skill、MCP、Tool 或 Lua capability 的入口、参数、权限和运行约束。它们只声明能力，不绕过 Adapter manifest 或全局 policy。

## English

Capability definitions describe entry points, parameters, permissions, and runtime constraints for Skills, MCP, Tools, or Lua capabilities. They declare capabilities only and do not bypass Adapter manifests or global policies.
