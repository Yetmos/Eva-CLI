# Routes / 路由配置

## 中文

Topic route 只配置事件投递，不包含业务逻辑。`target` 直接路由优先于 routes，`**` 通配符只能出现在 Topic pattern 的最后一段。

## English

Topic routes configure event delivery only; they do not contain business logic. Direct `target` routing takes precedence over routes, and `**` may appear only as the final segment of a Topic pattern.
