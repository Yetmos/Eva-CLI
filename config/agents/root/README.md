# Root Agent / 根 Agent

## 中文

`root-agent` 是顶层路由和任务拆解示例 Agent。它订阅 `/sys`，可以通过受控工具访问记忆、知识库和经过授权的 Adapter capability。

## English

`root-agent` is the sample top-level routing and task planning Agent. It subscribes to `/sys` and may use controlled tools for memory, knowledge, and authorized Adapter capabilities.
