# Route A Agent / Route A 智能体

## 中文

`agent-a` 是 `root-agent` 下的路由 Agent 示例，订阅 `/sys/route-a`，并通过 `routes.route-aa` 表达对子 Agent 的可读路由意图。

## English

`agent-a` is a sample route Agent under `root-agent`. It subscribes to `/sys/route-a` and uses `routes.route-aa` to document the intended child-Agent route.
