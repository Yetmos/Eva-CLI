# Route AA Children / Route AA 子智能体

## 中文

本目录保存 `agent-a` 的子 Agent 示例。目录嵌套只用于维护便利，运行时投递仍以 `agent.yaml` 和 `config/routes/topics.yaml` 为准。

## English

This directory contains sample child Agents for `agent-a`. Nesting is for maintainability only; runtime delivery is still controlled by each `agent.yaml` and `config/routes/topics.yaml`.
