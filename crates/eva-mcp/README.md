# eva-mcp / MCP 协议边界

## 中文

`eva-mcp` 负责 MCP client/server 协议、tool mapping、MCP policy helper 和 schema。它不把内部 Topic 无限制代理给外部 MCP client。

## English

`eva-mcp` owns MCP client/server protocol, tool mapping, MCP policy helpers, and schemas. It does not proxy internal Topics to external MCP clients without limits.
