# eva-config/src / 配置源码

## 中文

这里保存主配置、manifest、policy document、route table 和 schema 路径加载归一化逻辑。配置模块只产出可校验结构，不做最终权限裁决或运行时 mutation。

## English

This directory contains main configuration, manifest, policy document, route table, and schema path loading/normalization logic. The config crate produces validated structures only; it does not make final authorization decisions or mutate runtime state.
