# eva-storage/src / 存储源码

## 中文

这里保存状态存储、持久事件日志、artifact store 和本地实现边界。它只负责持久化接口和恢复数据，不承载 Agent 业务逻辑。

## English

This directory contains state storage, durable event log, artifact store, and local implementation boundaries. It owns persistence interfaces and recovery data only, not Agent business logic.
