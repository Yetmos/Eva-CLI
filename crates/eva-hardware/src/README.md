# eva-hardware/src / 硬件源码

## 中文

这里保存设备发现、registry、driver binding、热插拔和硬件状态。Lua raw I/O 必须保持关闭，硬件访问通过受控 Adapter 暴露。

## English

This directory contains device discovery, registry, driver binding, hotplug, and hardware state. Lua raw I/O must remain closed; hardware access is exposed through controlled Adapters.
