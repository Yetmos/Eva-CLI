# eva-adapter / 外部能力适配

## 中文

`eva-adapter` 负责 Adapter manifest、Registry、Router 和 transport runtime。它不做 Discovery 扫描，也不改写权限；只能接收 effective policy 并执行约束。

## English

`eva-adapter` owns Adapter manifests, Registry, Router, and transport runtimes. It does not perform Discovery scanning or rewrite permissions; it receives effective policy and enforces it.
