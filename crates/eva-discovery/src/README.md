# eva-discovery/src / 发现源码

## 中文

这里保存受信来源扫描、归一化、健康探测和缓存逻辑。Discovery 只返回候选项和拒绝原因，不授予运行时执行权。

## English

This directory contains trusted source scanning, normalization, health probing, and cache logic. Discovery returns candidates and rejection reasons only; it does not grant runtime execution authority.
