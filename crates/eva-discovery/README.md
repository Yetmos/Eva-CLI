# eva-discovery / 能力发现

## 中文

`eva-discovery` 负责受信来源扫描、归一化、健康探测和 cache。它不能授予执行权，只能返回 discovered candidate 和 rejected reason。

## English

`eva-discovery` owns trusted source scanning, normalization, health probing, and cache. It must not grant execution authority; it only returns discovered candidates and rejected reasons.
