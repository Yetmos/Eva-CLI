# eva-agent/src / Agent 运行源码

## 中文

这里保存 Agent 生命周期、队列、状态和事件处理边界。Lua 执行细节归 `eva-lua-host`，外部 provider transport 归 `eva-adapter`。

## English

This directory contains Agent lifecycle, queues, state, and event handling boundaries. Lua execution details belong in `eva-lua-host`, and external provider transports belong in `eva-adapter`.
