# eva-capability/src / 能力源码

## 中文

这里保存 capability registry、router、generation swap 和 host API trait。具体 provider、硬件 driver 或 MCP 实现应留在对应 Adapter 或 service crate。

## English

This directory contains the capability registry, router, generation swap, and host API traits. Concrete providers, hardware drivers, or MCP implementations belong in the corresponding Adapter or service crate.
